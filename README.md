# Santhe Connect 🌾📍

Santhe Connect is an Android application developed to help users discover and explore Karnataka’s traditional weekly markets (Santhe) through a modern digital platform. The application provides real-time market information, market schedules, specialties, search and filtering systems, favorites management, notifications, and interactive user experiences using modern Android technologies.

The application is designed using Jetpack Compose with a clean and responsive user interface and follows the MVVM architectural pattern for scalable and maintainable development. Firebase Firestore is integrated as the backend cloud database to provide real-time synchronization and data management.

# 📄 Problem Statement

Traditional weekly markets (Santhe) in Karnataka play an important role in local trade, agriculture, and cultural activities. However, information related to these markets is often scattered, unavailable online, or difficult for users to access. People face challenges in finding accurate market locations, schedules, specialties, and updates regarding different Santhe markets.

There is no centralized digital platform that provides organized and real-time information about Karnataka’s weekly markets. This creates difficulties for local vendors, customers, tourists, and small businesses in exploring and benefiting from these markets effectively.

To solve this problem, Santhe Connect was developed as a smart Android application that digitally connects users with traditional weekly markets using modern mobile technologies and cloud-based services.
